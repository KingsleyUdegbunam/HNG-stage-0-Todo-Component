# Todo Card

A single modern Todo / Task card component.

Built for Stage 0 task. Frontend track

## Requirement
- testability 
- accessibility, 
- responsiveness. 
 
 The result is a focused piece of UI that tracks a deadline in real time, responds to keyboard navigation, and degrades gracefully before JavaScript loads.

## What it does

- Counts down to a due date in plain language — *"5 days left"*, *"Due in 3 hrs"*, *"Overdue 2 mins"*
- Updates every 60 seconds, stops automatically after 3 days overdue
- Toggling the checkbox strikes the title, flips the status, and hides the countdown
- Edit and Delete buttons are keyboard-navigable with visible focus states
